What is HTML?
HTML stands for HyperText Markup Language and is used to structure content on the web.
What is CSS?
CSS stands for Cascading Style Sheets and controls the style and layout of web pages.
What is Bootstrap?
Bootstrap is a CSS framework for creating responsive, mobile-first websites with ready-made components.
What is JavaScript?
JavaScript is a scripting language used to create interactive elements on web pages.
What is React JS?
React JS is a JavaScript library for building user interfaces using reusable components.
What is Node.js?
Node.js is a runtime environment that lets developers run JavaScript on the server side.
What is Git?
Git is a version control system that tracks changes in source code during software development.
